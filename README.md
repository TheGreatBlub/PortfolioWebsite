# My Portfolio Website

A beginner full-stack portfolio project built while completing the Codecademy Full-Stack Engineer career path.

## Overview

This project is a personal portfolio website designed to showcase projects, practice frontend development skills, and experiment with interactive UI ideas.

The site includes:

* Responsive layout
* Hero section
* Project showcase area
* Dynamic project cards
* Local storage support
* File/image uploads for projects
* Interactive card hover effects

## Built With

* HTML5
* CSS3
* JavaScript
* Git & GitHub
* GitHub Pages

## Features

* Add projects dynamically using a form
* Upload custom project images
* Projects persist using localStorage
* Animated overlapping project cards
* Responsive design for smaller screens

## Goals of This Project

* Practice semantic HTML structure
* Improve CSS layout skills using Flexbox
* Learn DOM manipulation with JavaScript
* Use Git for version control
* Deploy a live website using GitHub Pages

## Future Improvements

* Better animations and transitions
* Real project data
* Improved mobile responsiveness
* Custom domain
* Dark/light theme toggle
* Backend integration
